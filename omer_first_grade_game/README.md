# מסע ההרים של עומר 🚗⛰️

משחק לימודי בעברית להכנה לכיתה א'. הילד/ה נוסע/ת עם מכונית על מפת ישראל, מהר החרמון בצפון ועד הרי אילת בדרום, וכל הר פותח סדרת תרגילים אינטראקטיביים.

## מה כולל המשחק

- **מפת ישראל אינטראקטיבית** עם 7 הרים, מכונית מונפשת ופתיחת שלבים הדרגתית.
- **7 סוגי מיני-משחקים:**
  - זיהוי אות פותחת (`letter_rec`)
  - ספירה וכמויות (`counting`)
  - כתיבת אותיות דפוס על קנבס (`letter_tracing`)
  - חיבור וחיסור (`math`)
  - הרכבת מילים / אות חסרה (`word_builder`)
  - השלמת סדרות (`sequence`)
  - הר האלופים המסכם (`champion_mix`)
- **3 דרגות קושי** (קל / בינוני / מאתגר).
- **הקראה קולית בעברית** דרך Web Speech API של הדפדפן — ללא שרת חיצוני.
- **אפקטי סאונד** מבוססי Web Audio API, קונפטי, רמזים ופידבק מעודד (ללא תחרותיות).
- **שמירת התקדמות** אוטומטית ב-localStorage.

## הרצה

```bash
npm install
npm run dev
```

פתחו את הכתובת שמופיעה בטרמינל (בדרך כלל http://localhost:5173).

## בנייה לפרודקשן

```bash
npm run build
npm run preview
```

## הערה על ההקראה הקולית

הגרסה הזו משתמשת ב-`window.speechSynthesis` (Web Speech API), כך שההקראה עובדת בכל דפדפן מודרני
ובכל מערכת הפעלה — ללא צורך בשרת TTS או ב-`say` של macOS. איכות הקול העברי תלויה בקולות
המותקנים במערכת/דפדפן של המשתמש.
